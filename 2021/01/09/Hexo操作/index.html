4、换主题（换成next）

在git bash操作 ，记得要在blog文件夹中操作，结束后会发现theme文件夹下面多了一个next主题

git clone https://github.com/iissnan/hexo-theme-next themes/next

先将blog根目录下的_config.yml文件中的theme字段的landscape改为next；

接着打开next文件夹下面的_config.yml文件，找到scheme字段，选择Pisces Scheme\Muse\Mist\Gemini（二任选其一）。

上传三步骤(记得是在blog根目录下用git bash运行)
